# Nature Guardians — Nonprofit environmental organization website
 
A responsive and user-friendly website for a nonprofit environmental organization. Users can learn about the mission, explore ongoing projects, read blog posts and news, discover upcoming events, and make donations to support the cause.
 
## 🔗 Live Demo
 
Experience the live website: [**Nature Guardians**](https://nature-guardians.vercel.app/)
 
---
 
## 🖥️ Technologies Used
 
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
 
---
 
## 🚀 Getting Started
 
To set up the project locally, run the following commands:
 
### 1. Clone the repository
 
```bash
git clone https://github.com/Kirill1908/Nature-Guardians.git
cd Nature-Guardians
```
 
### 2. Install dependencies
 
```bash
npm install
# or
yarn install
```
 
### 3. Run the project locally
 
```bash
npm run dev
# or
yarn dev
```
 
Open [http://localhost:3000](http://localhost:3000) in your browser.

## ✨ Features
 
- **Home Page** — hero section with a call to action and overview of the organization's mission
- **About Us** — team introduction, awards, and how donations are spent
- **What We Do** — description of environmental programs and initiatives
- **Media** — latest news and upcoming events
- **Contact Us** — contact form for reaching out to the organization
- **Donation** — options and instructions on how to contribute financially
- **Blog** — articles and posts on environmental topics
- **Events** — overview of upcoming and past events with details
- Fully responsive design for desktop, tablet, and mobile devices
